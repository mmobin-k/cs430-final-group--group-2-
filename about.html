What is KanFlow?

KanFlow is an interactive Kanban board web app built for the CS 430 Internet Multimedia Programming semester group project. It uses HTML5, CSS3, and vanilla JavaScript with Firebase Auth and Firestore for authentication and real-time cloud data storage.

Tasks live across five workflow columns - Backlog, To Do, In Progress, Review, and Done. Every drag, edit, and delete syncs live to Firestore, so changes appear instantly across all open sessions.
